# Available Attributes

Below is a list of available attributes that can be requested when a user validates via login.gov:

|----------------|------------------|
| Default?       |                  |
| Yes            | first_name       |
| Yes            | first_name       |
| Yes            | middle_name      |
| Yes            | last_name        |
| Yes            | address1         |
| Yes            | address2         |
| Yes            | city             |
| Yes            | state            |
| Yes            | zipcode          |
| Yes            | dob              |
| Yes            | ssn              |
| Yes            | phone            |
| No             | email            |
| No             | mobile           |
